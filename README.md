# Andrew Hollingworth — Portfolio

Personal portfolio site. Live at: https://www.andrewhollingworth.com

## Tech Stack

- [Vite](https://vite.dev/) + [React 18](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [React Router v6](https://reactrouter.com/)
- [Iconify](https://iconify.design/) for icons
- [Baskervville](https://fonts.google.com/specimen/Baskervville) (body) + [Satoshi](https://fontshare.com/fonts/satoshi) (headings)

## Local Development

```bash
cd client
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Build

```bash
npm run build      # TypeScript check + production build → client/dist/
npm run preview    # Serve the production build locally
```
